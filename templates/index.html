Overview

Prominence is a platform which allows users to easily exploit idle cloud resources for running scientific workloads. By default resources from the EGI Cloud Compute service are used.



Features



Run jobs from anywhere

Manage jobs using a web UI, a simple batch system style command line interface or interact programmatically using a RESTful API.



Reliability & reproduceability

All jobs are run in containers to ensure they will can run reliably anywhere and are reproducible.



Invisible infrastructure

All infrastructure provisioning is handled completely automatically and is totally invisible to the user.





MPI ready

Run multi-node Open MPI, Intel MPI or MPICH jobs in addition to single-node jobs.



Workflows

Construct complex workflows by specifying the dependencies between different jobs.



Data access

Jobs can access data transparently from OneData, WebDAV or object storage.




Scale from zero and back again

Resources are only used when they are needed to run workloads.



Submit globally, run globally

Submit from anywhere, run on any cloud.
